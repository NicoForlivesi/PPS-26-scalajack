# Report

Questa sezione documenta il processo di sviluppo del progetto ScalaJack, seguendo le fasi indicate
nella metodologia del corso: dai requisiti raccolti a inizio progetto, fino al design, all'implementazione
e alla verifica tramite test. Ogni sottosezione è mantenuta aggiornata parallelamente all'evoluzione del codice.
